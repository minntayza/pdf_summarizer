# Sketch Manifest

## Design Direction
Modern premium — sleek, polished like Notion/Linear, with bold personality like Arc/Raycast.
Sharp edges, subtle depth, distinctive accent colors, refined micro-interactions.
Black + warm gold accent, clean typography hierarchy, generous whitespace.
Everything elevated equally across upload, library, view, and study flows.

## Reference Points
- Notion (clean layout, subtle depth, professional polish)
- Linear (sharp edges, monochrome + accent, fast feel)
- Arc browser (bold color blocks, playful transitions, distinctive personality)
- Raycast (floating UI, command-bar aesthetic, refined micro-interactions)
