# Sketch 003: Navigation Chrome

## Design Question
How should the shared navigation and control bar feel? Every page shares this chrome — it needs to be consistent, unobtrusive, and functional. Controls: streak badge, font size toggle, theme toggle, language toggle, profile.

## How to View
open .planning/sketches/003-navigation-chrome/index.html

## Variants
- **A: Refined Top Bar** — Frosted glass sticky header with logo, nav links, and controls. Professional and familiar. Stays out of the way with subtle blur.
- **B: Minimal Header** — Ultra-thin sticky bar (44px). Smaller fonts, tighter spacing. Maximum content space. Feels fast and lightweight.
- **C: Floating Pill** — No top bar. Navigation floats at the bottom as a rounded pill (like Arc's command bar). Bold, distinctive, mobile-friendly. Most personality.

## What to Look For
- Content space vs chrome weight
- Frosted glass effect quality
- Control accessibility (font size, theme, language, profile)
- How it feels on scroll
- Mobile potential
